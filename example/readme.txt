# Mavisp Allosigma2 workflow - Example	
Shows run examples of the Mavisp Allosimga2 workflow using the same input protein for all test cases: FLG (Uniprot ID: P20930)

# Structure
- `protein_test/` 
	Step-by-step example of the full pipeline.
- `sidechain_value/` 
	Example run of step 4.allosigma_filtering, using `sidechain_value` distance filtering method.
- `sidechain_fraction/` 
	Example run of step 4.allosigma_filtering, using `sidechain_fraction` distance filtering method.

#Each folder contains its own `input`, `output`, `do.sh` and `readme.txt`
#The structure mirrors the main pipeline
